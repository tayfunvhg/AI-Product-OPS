# AI-Product-OPS

Репозиторий AI-агентов для продуктового офиса. Каждый агент решает конкретную задачу аналитика или продакт-менеджера и доступен как самостоятельное приложение.

---

## Агенты

### 1. [Classificator](./Classificator) — Классификатор инициатив

**Что делает:** Определяет, является ли инициатива **Продуктом** или **Проектом**, и выдаёт обоснованное заключение с процентной оценкой.

**Зачем это нужно:** Продукт и проект требуют разных моделей управления. Ошибка в классификации приводит к неверному выбору команды, процессов и критериев успеха. Агент помогает сделать этот выбор осознанно.

**Как работает:**
- Принимает материалы инициативы (презентация, описание, шаблон) или ведёт диалог с нуля
- Задаёт уточняющие открытые вопросы по 8 ключевым критериям: стратегический фокус, измерение успеха, источник требований, формирование бэклога, приоритезация, процесс, модель управления, горизонт
- Отслеживает уровень неопределённости и не делает вывод раньше времени
- Выдаёт двухшаговое заключение: сначала сводка фактов на подтверждение, затем скоринг + классификация + рекомендации

**Вывод:** скоринг в % · классификация (Продукт / Предварительно продукт / Неопределённо / Проект) · рекомендации по усилению модели управления

**Стек:** Python · Anthropic SDK (Claude) · Streamlit

```bash
cd Classificator
pip install -r requirements.txt
streamlit run app.py
```

### 2. [Vision Creator](./Vision/VisionCreator) — Формирование продуктового видения

**Что делает:** Через итеративный диалог помогает Product Owner сформулировать структурированное продуктовое видение — компактное, с фокусом на аудиторию, ценность и стратегическое направление.

**Зачем это нужно:** Размытое или отсутствующее видение — главная причина, по которой команды работают без ориентира. Агент задаёт правильные вопросы, отличает цели от метрик, стратегию от roadmap, и собирает результат в готовый документ.

**Как работает:**
- Принимает материалы (описание, презентации, метрики) или работает с нуля, а также дорабатывает существующее видение
- Отслеживает неопределённость по 12 разделам и не предлагает генерировать результат раньше времени
- Проверяет здравый смысл: если идея нереалистична или противоречива — мягко уточняет
- Генерирует видение только при неопределённости ≤ 0.01 (или ≤ 0.05 по запросу PO)

**Вывод:** формулировка видения · стратегия по годам с ориентирами · бизнес-цели с цифрами · контекст и риски

**Стек:** Python · Anthropic SDK (Claude) · Streamlit

```bash
cd Vision/VisionCreator
pip install -r requirements.txt
streamlit run app.py
```
