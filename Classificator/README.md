# Initiative Classifier v10

Агент-классификатор инициатив: **Продукт** или **Проект**.

Через итеративный диалог анализирует инициативу по 8 критериям и выдаёт обоснованное заключение с процентной оценкой.

## Стек

- **Python 3.11+**
- **Anthropic SDK** — Claude как движок агента
- **Streamlit** — веб-интерфейс чата
- **python-dotenv** — управление API-ключом

## Быстрый старт

```bash
# 1. Клонировать репозиторий и перейти в папку
cd Classificator

# 2. Установить зависимости
pip install -r requirements.txt

# 3. Создать .env файл с API-ключом
cp .env.example .env
# Вставить ваш Anthropic API Key в .env

# 4. Запустить
streamlit run app.py
```

Откроется браузер на `http://localhost:8501`.

Если `.env` не создавать — можно вставить API-ключ прямо в боковую панель приложения.

## Получить API-ключ

1. Зарегистрироваться на [console.anthropic.com](https://console.anthropic.com)
2. Создать API Key в разделе **API Keys**
3. Вставить в `.env` или в интерфейс приложения

## Структура

```
Classificator/
├── app.py            # Streamlit UI (запускать отсюда)
├── agent.py          # Логика агента и работа с Claude API
├── system_prompt.py  # Системный промпт агента
├── requirements.txt  # Зависимости
├── .env.example      # Пример файла с ключом
└── README.md
```

## Как пользоваться

1. Запустить приложение
2. Вставить описание/презентацию инициативы в чат
3. Отвечать на вопросы агента
4. Получить заключение: скоринг в % + классификация + рекомендации
