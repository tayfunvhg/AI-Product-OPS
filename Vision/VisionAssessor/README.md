# Vision Assessor

Агент-оценщик продуктового видения. Принимает готовый документ и выдаёт структурированный отчёт по 6 критериям с оценкой ИЗИ.

## Стек

- **Python 3.11+**
- **Anthropic SDK** — Claude как движок агента
- **Streamlit** — веб-интерфейс с текстовым полем ввода
- **python-dotenv** — управление API-ключом

## Быстрый старт

```bash
cd Vision/VisionAssessor
pip install -r requirements.txt
cp .env.example .env
# Вставить Anthropic API Key в .env
streamlit run app.py
```

## Как пользоваться

1. Вставить текст видения целиком в поле ввода (формулировку, стратегию, цели, контекст)
2. Нажать «Оценить»
3. Получить отчёт

## Что на выходе

Отчёт по 6 критериям с оценкой Ok / Частично / Не проходит:

1. *Структурная целостность* — длина, аудитория, ясность формулировки
2. *Временной горизонт* — наличие явного горизонта 3+ лет
3. *Клиентоцентричность* — аудитория и измеримая ценность для неё
4. *Ценность для бизнеса* — внутренний эффект: автоматизация, снижение издержек и т.д.
5. *Цели и метрики* — измеримые долгосрочные цели с цифрами и сроками
6. *Стратегия* — связь с ориентирами: цифровизация, инновации, лидерство и т.д.

Итоговая оценка ИЗИ (0-2) и вердикт: *Готово к использованию / Нужна доработка / Требует фундаментальной переработки*.
