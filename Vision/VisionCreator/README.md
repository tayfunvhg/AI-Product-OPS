# Vision Creator

Агент-стратег, который через итеративный диалог помогает Product Owner сформулировать структурированное продуктовое видение.

## Стек

- **Python 3.11+**
- **Anthropic SDK** — Claude как движок агента
- **Streamlit** — веб-интерфейс чата
- **python-dotenv** — управление API-ключом

## Быстрый старт

```bash
cd Vision/VisionCreator
pip install -r requirements.txt
cp .env.example .env
# Вставить Anthropic API Key в .env
streamlit run app.py
```

Откроется браузер на `http://localhost:8501`.

## Три способа начать

1. **С нуля** — рассказать про продукт, агент задаст уточняющие вопросы
2. **С материалами** — прислать описание, презентации, метрики — агент проанализирует и спросит по пробелам
3. **Доработка** — прислать текущее видение, агент оценит и предложит улучшения

## Что на выходе

Структурированный документ в трёх блоках:

- **Блок 1** — формулировка видения (10-25 слов) + конкретная измеримая ценность для пользователей
- **Блок 2** — стратегия по годам на горизонте 3-5 лет, каждый шаг привязан к стратегическому ориентиру
- **Блок 3** — ключевые бизнес-цели с целевыми значениями и сроками
- **Контекст** (опционально) — текущая ситуация, допущения, риски, открытые вопросы
