# Hypothesis Generator

Агент-генератор продуктовых гипотез. Собирает контекст о продукте и выдаёт структурированный набор гипотез с обоснованием, уровнем уверенности и рейтингом.

## Стек

- **Python 3.11+**
- **Anthropic SDK** — Claude как движок агента
- **Streamlit** — чат-интерфейс
- **python-dotenv** — управление API-ключом

## Быстрый старт

```bash
cd Discovery/Hyp.Generator
pip install -r requirements.txt
cp .env.example .env
# Вставить Anthropic API Key в .env
streamlit run app.py
```

## Как пользоваться

1. Загрузить материалы: видение, брифы, метрики, исследования, бэклог (в любом формате)
2. Если материалов нет — агент задаст вопросы сам
3. После заполнения минимум 2 блоков контекста — агент генерирует 5-10 гипотез
4. Обсудить, расставить приоритеты, получить финальный артефакт

## Формат гипотезы

```
Если мы [действие], то [метрика] изменится в сторону [направление]
[ДАННЫЕ ТРЕБУЮТСЯ / с X до Y (из [источник])],
потому что [обоснование (из [источник])].
```

## Что на выходе

Артефакт из 7 блоков:
1. Контекст (источники и ключевые факты)
2. Заполненные блоки контекста
3. Гипотезы с уверенностью и обоснованием
4. Рейтинг гипотез
5. Пробелы в данных (что нужно собрать)
6. Рекомендуемые следующие шаги
7. Что не вошло в гипотезы
