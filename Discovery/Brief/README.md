# Brief Writer

Агент для написания продуктового брифа. Через итеративный диалог собирает всю необходимую информацию об инициативе и формирует готовый документ для передачи в команду.

## Стек

- **Python 3.11+**
- **Anthropic SDK** — Claude как движок агента
- **Streamlit** — чат-интерфейс
- **python-dotenv** — управление API-ключом

## Быстрый старт

```bash
cd Discovery/Brief
pip install -r requirements.txt
cp .env.example .env
# Вставить Anthropic API Key в .env
streamlit run app.py
```

## Как пользоваться

1. Описать инициативу своими словами или загрузить любые материалы
2. Агент задаёт уточняющие вопросы по 7 критическим и 5 вспомогательным разделам
3. При неопределённости ≤ 10% агент показывает сводку фактов на подтверждение
4. После подтверждения — формирует полный бриф

## Что на выходе

Полный продуктовый бриф из 12 разделов:
1. Что делаем (скоуп + Job Story)
2. Зачем (полезное действие)
3. Для кого (пользователи и боли)
4. Текущая ситуация
5. Критерии приёмки (чеклист)
6. Вне скоупа
7. Жёсткие ограничения
8. Допущения и зависимости
9. Необходимая информация
10. Приоритизация RICE
11. Метрики успеха
12. Риски
